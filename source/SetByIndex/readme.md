## Set by Index

**Set by Index** quickly generates Print Sets based on an existing Sheet Schedule.

### 🚀 Usage

Select a Sheet Schedule within your project, and the tool will automatically create a corresponding Print Set containing only those sheets.

### ⚠️ Note

This tool is currently considered **Legacy**. Since recent Revit versions now include native functionality to save print selections directly from the browser, this utility is primarily maintained for compatibility with older workflows or rapid set generation from filtered schedules.
