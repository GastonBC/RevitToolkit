## Element View Range

**Element View Range** instantly aligns a plan’s View Range to match a selected object. It eliminates the manual math of calculating offsets for Floor or Ceiling plans.

### How it works

1. **Select an element** (or draw a 2D Detail Line) at the desired height.
2. **Run the tool.**
3. The View Range (Cut Plane and Top/Bottom) is automatically adjusted to match.

### 💡 Pro Tip

Using **2D Lines** is the most efficient way to use this tool. You can draw a line exactly where you want the cut to happen; the tool will snap the View Range to that elevation and **automatically delete the line** for you, keeping your view clean.
Recommended shortcut: EVR
