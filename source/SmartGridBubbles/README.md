# Smart Grid Bubbles

**Smart Grid Bubbles** is a productivity tool designed to standardize grid visibility instantly. Instead of manually clicking individual checkboxes for every grid line in a view, this tool automates the process to follow standard drafting conventions.

### How it works

1. **Select** the grid lines you want to modify in your current view.
2. **Click** the Smart Grid Bubbles button.
3. The tool automatically:
* **Enables** bubbles for the **Top** and **Left** ends.
* **Disables** bubbles for the **Bottom** and **Right** ends.


### 💡 Why use it?

Manual grid management is tedious and prone to human error. This tool ensures that your project views remain consistent and professional with a single click, saving time during sheet setup and documentation.
